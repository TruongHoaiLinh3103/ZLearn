### How to use
1. Click "**Use this template**" or clone this repository.
2. cd repository.
3. "**Use Terminal**" cd "**server**" and "**npm i**" => "**npm start**" (note: Must have a database with the name: "**product**").
4. "**Use another terminal**" cd "**client**" and "**npm i**" => "**npm start**".
